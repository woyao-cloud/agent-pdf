# 第40章 插件系统与扩展机制

FastAPI 的扩展机制让应用具备可插拔的能力。本章从生命周期钩子到自定义扩展，从事件系统到插件架构，深入剖析 FastAPI 插件系统的机制与实践。

## 40.1 概念与语法

### 40.1.1 生命周期钩子

```python
from contextlib import asynccontextmanager
from fastapi import FastAPI

# lifespan 钩子
@asynccontextmanager
async def lifespan(app: FastAPI):
    # Startup 钩子
    print("Starting up...")
    app.state.db = await connect_database()
    app.state.redis = await connect_redis()
    app.state.cache = await init_cache()
    yield
    # Shutdown 钩子
    print("Shutting down...")
    await app.state.db.close()
    await app.state.redis.close()
    await app.state.cache.clear()

app = FastAPI(lifespan=lifespan)

# on_event 钩子（已弃用，但仍可用）
@app.on_event("startup")
async def startup():
    print("Starting up...")

@app.on_event("shutdown")
async def shutdown():
    print("Shutting down...")
```

### 40.1.2 中间件扩展

```python
from fastapi import FastAPI, Request
from starlette.middleware.base import BaseHTTPMiddleware
import time
import uuid

class RequestIDMiddleware(BaseHTTPMiddleware):
    async def dispatch(self, request: Request, call_next):
        request_id = request.headers.get("X-Request-ID") or str(uuid.uuid4())
        request.state.request_id = request_id
        response = await call_next(request)
        response.headers["X-Request-ID"] = request_id
        return response

class TimingMiddleware(BaseHTTPMiddleware):
    async def dispatch(self, request: Request, call_next):
        start = time.time()
        response = await call_next(request)
        response.headers["X-Response-Time"] = f"{time.time() - start:.3f}s"
        return response

class ErrorHandlingMiddleware(BaseHTTPMiddleware):
    async def dispatch(self, request: Request, call_next):
        try:
            return await call_next(request)
        except Exception as e:
            import logging
            logging.exception(f"Unhandled error: {e}")
            from fastapi.responses import JSONResponse
            return JSONResponse(status_code=500, content={"detail": "Internal server error"})

app = FastAPI()
app.add_middleware(ErrorHandlingMiddleware)
app.add_middleware(TimingMiddleware)
app.add_middleware(RequestIDMiddleware)
```

### 40.1.3 自定义扩展类

```python
from fastapi import FastAPI
from typing import Callable

class Plugin:
    """插件基类。"""
    def __init__(self, app: FastAPI):
        self.app = app

    def on_startup(self):
        """启动时执行。"""
        pass

    def on_shutdown(self):
        """关闭时执行。"""
        pass

    def add_middleware(self):
        """添加中间件。"""
        pass

    def add_routes(self):
        """添加路由。"""
        pass

class MetricsPlugin(Plugin):
    """指标收集插件。"""
    def __init__(self, app: FastAPI, prefix: str = "/metrics"):
        super().__init__(app)
        self.prefix = prefix
        self.request_count = 0
        self.request_times: list[float] = []

    def add_routes(self):
        @self.app.get(self.prefix)
        async def metrics():
            return {
                "request_count": self.request_count,
                "avg_response_time": sum(self.request_times) / max(len(self.request_times), 1),
            }

    def add_middleware(self):
        @self.app.middleware("http")
        async def metrics_middleware(request, call_next):
            self.request_count += 1
            start = time.time()
            response = await call_next(request)
            self.request_times.append(time.time() - start)
            return response

# 使用插件
app = FastAPI()
metrics = MetricsPlugin(app)
metrics.add_routes()
metrics.add_middleware()
```

### 40.1.4 路由级别扩展

```python
from fastapi import APIRouter, Depends, FastAPI

# 可复用的依赖模块
class AuthModule:
    def __init__(self, secret_key: str):
        self.secret_key = secret_key

    def get_current_user(self):
        async def dependency(token: str = Header(...)):
            user = verify_token(token, self.secret_key)
            if not user:
                raise HTTPException(401)
            return user
        return dependency

    def require_role(self, role: str):
        async def dependency(user=Depends(self.get_current_user())):
            if user.role != role:
                raise HTTPException(403)
            return user
        return dependency

# 模块化路由
class UserModule:
    router = APIRouter(prefix="/users", tags=["users"])

    def __init__(self, auth: AuthModule):
        self.auth = auth

    def register(self, app: FastAPI):
        @self.router.get("/me")
        async def get_me(user=Depends(self.auth.get_current_user())):
            return user

        @self.router.get("/admin/dashboard")
        async def admin_dashboard(user=Depends(self.auth.require_role("admin"))):
            return {"message": "Admin dashboard"}

        app.include_router(self.router)

# 组装应用
app = FastAPI()
auth = AuthModule(secret_key="secret")
user_module = UserModule(auth)
user_module.register(app)
```

## 40.2 原理与机制

### 40.2.1 FastAPI 扩展点

```python
"""
FastAPI 的核心扩展点：

1. 生命周期（Lifespan）：
   - @asynccontextmanager
   - startup/shutdown 钩子
   - app.state 全局状态

2. 中间件（Middleware）：
   - @app.middleware("http")
   - BaseHTTPMiddleware
   - 纯 ASGI 中间件
   - 内置中间件（CORS、GZip、TrustedHost）

3. 依赖注入（Dependency Injection）：
   - Depends() 声明依赖
   - yield 依赖（资源管理）
   - app.dependency_overrides（测试覆盖）
   - 全局依赖（router 级别）

4. 路由（Routing）：
   - APIRouter 模块化
   - 自定义 APIRoute 类
   - 路由级依赖

5. 异常处理（Exception Handling）：
   - @app.exception_handler(CustomException)
   - 内置异常处理器（HTTPException、ValidationError）

6. 事件系统：
   - on_event("startup") / on_event("shutdown")
   - 中间件中的请求级事件

7. OpenAPI 自定义：
   - app.openapi = custom_openapi
   - 自定义 Schema 生成
   - 文档 URL 配置
"""
```

### 40.2.2 插件架构模式

```python
"""
插件架构模式：

1. 注册模式（最简单）：
   plugin = Plugin(app)
   plugin.register()
   → 插件在 register() 中注册路由和中间件

2. 生命周期模式：
   class Plugin:
       async def on_startup(self): ...
       async def on_shutdown(self): ...

   → 统一在 lifespan 中调用所有插件的钩子

3. 事件总线模式：
   class EventBus:
       subscribers: dict[str, list[Callable]] = {}

       def subscribe(self, event: str, handler: Callable):
           self.subscribers[event].append(handler)

       async def publish(self, event: str, data: Any):
           for handler in self.subscribers[event]:
               await handler(data)

4. 装饰器模式：
   def plugin(name: str):
       def decorator(cls):
           cls.plugin_name = name
           return cls
       return decorator

   → 使用装饰器标记插件类

5. 配置驱动模式：
   PLUGINS = [
       {"name": "metrics", "config": {"prefix": "/metrics"}},
       {"name": "auth", "config": {"secret_key": "..."}},
   ]

   → 从配置加载和初始化插件
"""
```

## 40.3 使用场景

### 40.3.1 可插拔应用框架

```python
from contextlib import asynccontextmanager
from fastapi import FastAPI
from typing import Any, Callable

class PluginManager:
    """插件管理器。"""
    def __init__(self):
        self.plugins: list[Any] = []

    def register(self, plugin_class, **config):
        plugin = plugin_class(**config)
        self.plugins.append(plugin)
        return plugin

    async def on_startup(self, app: FastAPI):
        for plugin in self.plugins:
            if hasattr(plugin, "on_startup"):
                await plugin.on_startup(app)

    async def on_shutdown(self, app: FastAPI):
        for plugin in reversed(self.plugins):
            if hasattr(plugin, "on_shutdown"):
                await plugin.on_shutdown(app)

# 定义插件
class DatabasePlugin:
    def __init__(self, url: str = "sqlite:///app.db"):
        self.url = url
        self.engine = None

    async def on_startup(self, app: FastAPI):
        self.engine = create_async_engine(self.url)
        app.state.db_engine = self.engine
        print(f"Database connected: {self.url}")

    async def on_shutdown(self, app: FastAPI):
        await self.engine.dispose()
        print("Database disconnected")

class CachePlugin:
    def __init__(self, url: str = "redis://localhost:6379"):
        self.url = url
        self.redis = None

    async def on_startup(self, app: FastAPI):
        import redis.asyncio as aioredis
        self.redis = aioredis.from_url(self.url)
        app.state.redis = self.redis
        print(f"Redis connected: {self.url}")

    async def on_shutdown(self, app: FastAPI):
        await self.redis.close()
        print("Redis disconnected")

# 组装应用
plugin_manager = PluginManager()
plugin_manager.register(DatabasePlugin, url="postgresql+asyncpg://user:pass@localhost/db")
plugin_manager.register(CachePlugin, url="redis://localhost:6379")

@asynccontextmanager
async def lifespan(app: FastAPI):
    await plugin_manager.on_startup(app)
    yield
    await plugin_manager.on_shutdown(app)

app = FastAPI(lifespan=lifespan)
```

## 40.4 示例代码

### 40.4.1 完整的插件系统

```python
"""FastAPI 插件系统 — 可插拔的应用框架"""
from abc import ABC, abstractmethod
from contextlib import asynccontextmanager
from fastapi import FastAPI

class AppPlugin(ABC):
    """插件基类。"""
    @abstractmethod
    async def on_startup(self, app: FastAPI):
        pass

    @abstractmethod
    async def on_shutdown(self, app: FastAPI):
        pass

    def register_routes(self, app: FastAPI):
        pass

    def register_middleware(self, app: FastAPI):
        pass

class HealthPlugin(AppPlugin):
    """健康检查插件。"""
    async def on_startup(self, app: FastAPI):
        app.state.health_checks = {}

    async def on_shutdown(self, app: FastAPI):
        pass

    def register_routes(self, app: FastAPI):
        @app.get("/health")
        async def health():
            checks = app.state.health_checks
            results = {name: check() for name, check in checks.items()}
            status = "healthy" if all(results.values()) else "unhealthy"
            return {"status": status, "checks": results}

class MetricsPlugin(AppPlugin):
    """指标收集插件。"""
    def __init__(self, prefix: str = "/metrics"):
        self.prefix = prefix
        self.request_count = 0

    async def on_startup(self, app: FastAPI):
        app.state.metrics = self

    async def on_shutdown(self, app: FastAPI):
        pass

    def register_routes(self, app: FastAPI):
        @app.get(self.prefix)
        async def metrics():
            return {"request_count": self.request_count}

    def register_middleware(self, app: FastAPI):
        @app.middleware("http")
        async def metrics_middleware(request, call_next):
            self.request_count += 1
            return await call_next(request)

# 应用工厂
def create_app(plugins: list[AppPlugin] = None):
    plugins = plugins or []
    plugin_list = [p() if isinstance(p, type) else p for p in plugins]

    @asynccontextmanager
    async def lifespan(app: FastAPI):
        for plugin in plugin_list:
            await plugin.on_startup(app)
        yield
        for plugin in reversed(plugin_list):
            await plugin.on_shutdown(app)

    app = FastAPI(lifespan=lifespan)
    for plugin in plugin_list:
        plugin.register_routes(app)
        plugin.register_middleware(app)

    return app

# 使用
app = create_app(plugins=[HealthPlugin, MetricsPlugin(prefix="/api/metrics")])
```

## 40.5 常见陷阱与最佳实践

| 陷阱 | 最佳实践 |
|------|---------|
| 插件顺序依赖 | 明确定义插件依赖和加载顺序 |
| 插件间直接耦合 | 使用事件总线或 app.state 通信 |
| 不处理插件异常 | 插件 on_startup/on_shutdown 需 try/except |
| 全局状态污染 | 插件状态存储在 app.state.plugin_name |
| 不支持插件热插拔 | 设计插件卸载接口 |
| 中间件顺序混乱 | 明确文档说明中间件添加顺序 |
