# 第37章 自定义响应类与渲染器

FastAPI 支持多种响应类型和自定义渲染。本章从内置响应类到自定义渲染器，从流式响应到二进制文件，深入剖析 FastAPI 响应处理机制的底层原理。

## 37.1 概念与语法

### 37.1.1 内置响应类

```python
from fastapi import FastAPI
from fastapi.responses import (
    JSONResponse,
    HTMLResponse,
    PlainTextResponse,
    RedirectResponse,
    StreamingResponse,
    FileResponse,
    Response,
)

app = FastAPI()

# JSONResponse（默认）
@app.get("/json")
async def json_response():
    return JSONResponse(content={"message": "Hello"}, status_code=200)

# HTMLResponse
@app.get("/html", response_class=HTMLResponse)
async def html_response():
    return "<h1>Hello World</h1>"

# PlainTextResponse
@app.get("/text", response_class=PlainTextResponse)
async def text_response():
    return "Hello World"

# RedirectResponse
@app.get("/redirect")
async def redirect_response():
    return RedirectResponse(url="/json", status_code=301)

# StreamingResponse
@app.get("/stream")
async def stream_response():
    async def generate():
        for i in range(10):
            yield f"data: {i}\n\n"
    return StreamingResponse(generate(), media_type="text/event-stream")

# FileResponse
@app.get("/download/{filename}")
async def download_file(filename: str):
    return FileResponse(
        path=f"uploads/{filename}",
        filename=filename,
        media_type="application/octet-stream",
    )

# 自定义状态码和头
@app.get("/custom")
async def custom_response():
    return JSONResponse(
        content={"message": "Created"},
        status_code=201,
        headers={"X-Custom-Header": "value"},
    )
```

### 37.1.2 自定义响应类

```python
from fastapi import FastAPI
from fastapi.responses import JSONResponse
import orjson

# orjson 响应类（更快）
class ORJSONResponse(JSONResponse):
    def render(self, content: Any) -> bytes:
        return orjson.dumps(content)

app = FastAPI(default_response_class=ORJSONResponse)

@app.get("/fast-json")
async def fast_json():
    return {"message": "Fast JSON with orjson"}

# XML 响应类
class XMLResponse(Response):
    media_type = "application/xml"

    def render(self, content: Any) -> bytes:
        if isinstance(content, str):
            return content.encode("utf-8")
        return b"<error>Invalid content</error>"

@app.get("/xml", response_class=XMLResponse)
async def xml_response():
    return "<root><message>Hello</message></root>"

# CSV 响应类
class CSVResponse(Response):
    media_type = "text/csv"

    def render(self, content: Any) -> bytes:
        import csv
        import io
        output = io.StringIO()
        writer = csv.writer(output)
        for row in content:
            writer.writerow(row)
        return output.getvalue().encode("utf-8")

@app.get("/csv", response_class=CSVResponse)
async def csv_response():
    return [["id", "name"], [1, "Alice"], [2, "Bob"]]

# 设置默认响应类
app = FastAPI(default_response_class=ORJSONResponse)
```

### 37.1.3 流式响应

```python
from fastapi import FastAPI
from fastapi.responses import StreamingResponse
import asyncio
import io

app = FastAPI()

# SSE 流式响应
@app.get("/sse")
async def sse_stream():
    async def event_stream():
        for i in range(10):
            yield f"data: {{\"count\": {i}}}\n\n"
            await asyncio.sleep(1)

    return StreamingResponse(
        event_stream(),
        media_type="text/event-stream",
        headers={"Cache-Control": "no-cache", "X-Accel-Buffering": "no"},
    )

# 大文件流式下载
@app.get("/download-large")
async def download_large_file():
    async def file_stream():
        chunk_size = 64 * 1024  # 64KB
        with open("large_file.zip", "rb") as f:
            while chunk := f.read(chunk_size):
                yield chunk

    return StreamingResponse(
        file_stream(),
        media_type="application/zip",
        headers={"Content-Disposition": "attachment; filename=large_file.zip"},
    )

# 实时数据流
@app.get("/realtime")
async def realtime_data():
    async def generate_data():
        while True:
            data = await get_latest_data()
            yield f"data: {data}\n\n"
            await asyncio.sleep(0.5)

    return StreamingResponse(
        generate_data(),
        media_type="text/event-stream",
    )
```

## 37.2 原理与机制

### 37.2.1 Response 类的层次结构

```python
"""
FastAPI 响应类的层次结构：

Response（基类）
├── HTMLResponse
├── PlainTextResponse
├── JSONResponse
│   └── ORJSONResponse（自定义）
├── RedirectResponse
├── StreamingResponse
├── FileResponse
└── 自定义 Response 子类

Response 的核心属性和方法：
- media_type: 响应的 MIME 类型
- status_code: HTTP 状态码
- headers: 响应头字典
- body: 响应体字节

Response 的 ASGI 协议：
async def __call__(self, scope, receive, send):
    # 1. 发送 http.response.start
    await send({
        "type": "http.response.start",
        "status": self.status_code,
        "headers": self.encode_headers(),
    })
    # 2. 发送 http.response.body
    await send({
        "type": "http.response.body",
        "body": self.body,
    })

JSONResponse.render() 源码：
def render(self, content: Any) -> bytes:
    return json.dumps(content, ...).encode("utf-8")

StreamingResponse.__call__() 源码：
async def __call__(self, scope, receive, send):
    # 1. 发送响应头
    await send({"type": "http.response.start", ...})
    # 2. 流式发送响应体
    async for chunk in self.body_iterator:
        await send({"type": "http.response.body", "body": chunk, "more_body": True})
    # 3. 发送结束标记
    await send({"type": "http.response.body", "body": b"", "more_body": False})
"""
```

### 37.2.2 response_model 与 Response 类

```python
"""
response_model 和 Response 类的关系：

1. response_model 是 Pydantic 过滤器：
   - 路由函数返回的原始数据
   - 通过 response_model 过滤
   - 只包含 response_model 定义的字段
   - 序列化为 JSON

2. response_class 是响应格式化器：
   - 控制最终响应的格式
   - JSONResponse → JSON 格式
   - HTMLResponse → HTML 格式
   - StreamingResponse → 流式传输

3. 两者协作流程：
   路由函数返回数据
   → response_model 过滤
   → jsonable_encoder 序列化
   → response_class.render() 渲染
   → ASGI send() 发送

4. 优先级：
   - 如果路由函数直接返回 Response 对象（如 JSONResponse）
   - FastAPI 不经过 response_model 过滤
   - 直接使用返回的 Response 对象

源码位置：fastapi/routing.py → get_request_handler()
"""
```

### 37.3.1 图片处理与返回

```python
from fastapi import FastAPI
from fastapi.responses import Response
from PIL import Image
import io

app = FastAPI()

@app.get("/thumbnail/{image_id}")
async def get_thumbnail(image_id: int, size: int = 100):
    img = load_image(image_id)
    img.thumbnail((size, size))

    buffer = io.BytesIO()
    img.save(buffer, format="JPEG", quality=85)
    img_bytes = buffer.getvalue()

    return Response(
        content=img_bytes,
        media_type="image/jpeg",
        headers={"Cache-Control": "public, max-age=3600"},
    )
```

## 37.4 示例代码

### 37.4.1 完整的自定义响应体系

```python
"""自定义响应体系 — JSON、XML、CSV 多格式支持"""
from fastapi import FastAPI, Request
from fastapi.responses import JSONResponse, Response
from typing import Any
import orjson

app = FastAPI()

# orjson 响应
class ORJSONResponse(JSONResponse):
    def render(self, content: Any) -> bytes:
        return orjson.dumps(
            content,
            option=orjson.OPT_SERIALIZE_NUMPY | orjson.OPT_INDENT_2,
        )

# XML 响应
class XMLResponse(Response):
    media_type = "application/xml"
    def render(self, content: Any) -> bytes:
        if isinstance(content, str):
            return content.encode("utf-8")
        return b"<error>Invalid content type</error>"

# 内容协商中间件
@app.middleware("http")
async def content_negotiation(request: Request, call_next):
    response = await call_next(request)
    accept = request.headers.get("accept", "application/json")
    # 可根据 Accept 头调整响应格式
    return response

@app.get("/items/{item_id}")
async def get_item(item_id: int, format: str = "json"):
    data = {"id": item_id, "name": f"Item {item_id}"}
    if format == "xml":
        return XMLResponse(content=f"<item><id>{item_id}</id><name>Item {item_id}</name></item>")
    return ORJSONResponse(content=data)
```

## 37.5 常见陷阱与最佳实践

| 陷阱 | 最佳实践 |
|------|---------|
| 默认 JSON 序列化慢 | 使用 orjson 替代 json |
| 大文件全部读入内存 | 使用 StreamingResponse 流式传输 |
| 不设置 Content-Type | 所有响应设置正确的 media_type |
| 流式响应不设缓冲头 | SSE 设置 X-Accel-Buffering: no |
| FileResponse 路径遍历 | 验证文件名不含 ../ |
| response_model 与 Response 类冲突 | 直接返回 Response 时不受 response_model 影响 |
