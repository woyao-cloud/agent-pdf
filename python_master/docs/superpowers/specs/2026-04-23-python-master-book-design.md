# 精通Python - 书籍设计文档

## 项目概述

编写一本全面深入的Python书籍，涵盖从基础到极致深度的所有主要概念和高级概念。正文中文，技术术语保留英文。每章包含概念语法、原理机制(C源码级)、使用场景、完整示例代码、常见陷阱与最佳实践。

## 目标读者

全层次覆盖：初学者可从头读起，有经验者可跳读进阶章节。

## 组织方式

线性递进式：12大部分、56章，从基础到高级逐步推进。

## 每章统一结构

1. **概念与语法** — 是什么、怎么用
2. **原理与机制** — 底层工作原理（C源码级分析）
3. **使用场景** — 什么时候用、为什么用
4. **示例代码** — 可运行的完整示例
5. **常见陷阱与最佳实践**

## 输出格式

- 每章一个Markdown文件
- 文件命名：`PartXX_章节号_章节标题.md`（如 `Part01_01_Python概述与环境搭建.md`）
- 所有代码示例可独立运行
- 正文中文，技术术语保留英文

## 深度要求

极致深度：原理讲透、C源码级分析、性能对比、边界情况全覆盖。
