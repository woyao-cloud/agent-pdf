## 25.1 概念与语法

### 25.1.1 `==` 与 `is`

Python 有两种比较运算符：`==`（值相等）和 `is`（身份相同）。

| 运算符 | 含义 | 对应特殊方法 | C 层面 |
|--------|------|-------------|--------|
| `==` | 值相等 | `__eq__` | `PyObject_RichCompareBool(op, op2, Py_EQ)` |
| `is` | 身份相同 | 无 | 直接比较 `PyObject *` 指针值 |

```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)  # True  — 值相等
print(a is b)  # False — 不是同一个对象
print(a is c)  # True  — 是同一个对象

# is 比较的是 id()
print(id(a) == id(c))  # True
print(id(a) == id(b))  # False
```

**`is` 的典型用途**：

```python
# 1. 与 None 比较（PEP 8 推荐）
x = None
if x is None:     # 推荐
    pass
if x == None:     # 不推荐
    pass

# 2. 与布尔单例比较
if value is True:   # 推荐
    pass

# 3. 检查对象是否为特定哨兵值
MISSING = object()  # 创建唯一的哨兵对象

def get_value(key, default=MISSING):
    if default is MISSING:
        # 用户没有提供 default
        pass
    else:
        # 用户显式提供了 default（即使 default=None）
        pass
```

### 25.1.2 `__eq__` 协议

`==` 运算符调用 `__eq__` 方法。如果未定义 `__eq__`，默认比较对象身份（等价于 `is`）。

```python
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    # 不定义 __eq__ 时，== 等价于 is

p1 = Point(1, 2)
p2 = Point(1, 2)
print(p1 == p2)  # False — 默认比较身份

# 定义 __eq__ 后，== 比较值
class PointEq:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __eq__(self, other):
        if not isinstance(other, PointEq):
            return NotImplemented
        return self.x == other.x and self.y == other.y

p1 = PointEq(1, 2)
p2 = PointEq(1, 2)
print(p1 == p2)  # True — 值相等
print(p1 is p2)  # False — 仍是不同对象
```

**`__eq__` 的返回值规则**：

```python
class Good:
    def __eq__(self, other):
        if not isinstance(other, Good):
            return NotImplemented  # 不支持与其他类型比较时返回 NotImplemented
        return True

# NotImplemented 会让 Python 尝试反向比较：
# a == b → 如果 a.__eq__(b) 返回 NotImplemented
#        → 尝试 b.__eq__(a)
# 这称为"反射规则"（reflected operation）
```

### 25.1.3 `__hash__` 协议

`__hash__` 与 `__eq__` 必须协同工作。核心规则：**如果 `a == b`，则 `hash(a) == hash(b)`**。

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __eq__(self, other):
        if not isinstance(other, Person):
            return NotImplemented
        return self.name == other.name and self.age == other.age

    def __hash__(self):
        return hash((self.name, self.age))  # 基于 __eq__ 使用的相同字段

p1 = Person("Alice", 30)
p2 = Person("Alice", 30)

print(p1 == p2)          # True
print(hash(p1) == hash(p2))  # True — 哈希一致

# 现在可以安全地用作 dict key 和 set 元素
d = {p1: "engineer"}
print(d[p2])  # "engineer" — p2 能找到 p1 的值
```

**Python 3 的自动 `__hash__` 行为**：

```python
# 情况1：只定义 __eq__，__hash__ 自动设为 None（不可哈希）
class OnlyEq:
    def __init__(self, x):
        self.x = x

    def __eq__(self, other):
        return isinstance(other, OnlyEq) and self.x == other.x

obj = OnlyEq(1)
# hash(obj)  # TypeError: unhashable type: 'OnlyEq'

# 情况2：不定义 __eq__ 和 __hash__，使用默认行为
class Default:
    pass

obj = Default()
print(hash(obj))  # 基于对象 id 的哈希值

# 情况3：显式定义 __hash__ = None，明确不可哈希
class ExplicitUnhashable:
    __hash__ = None

# 情况4：同时定义 __eq__ 和 __hash__
class Full:
    def __init__(self, x):
        self.x = x

    def __eq__(self, other):
        return isinstance(other, Full) and self.x == other.x

    def __hash__(self):
        return hash(self.x)

obj = Full(1)
print(hash(obj))  # OK — 可哈希
```

### 25.1.4 浅拷贝与深拷贝

```python
import copy

# 浅拷贝：创建新对象，但元素仍引用原对象
original = [[1, 2], [3, 4]]
shallow = copy.copy(original)

shallow[0].append(3)  # 修改了内部的 list
print(original)  # [[1, 2, 3], [3, 4]] — 原对象也被修改！

# 深拷贝：递归拷贝所有嵌套对象
original = [[1, 2], [3, 4]]
deep = copy.deepcopy(original)

deep[0].append(3)  # 只修改了深拷贝
print(original)  # [[1, 2], [3, 4]] — 原对象不受影响

# 各类型的浅拷贝方式
lst = [1, 2, 3]
# 方式1：copy.copy()
c1 = copy.copy(lst)
# 方式2：切片
c2 = lst[:]
# 方式3：list()
c3 = list(lst)
# 方式4：copy 方法
c4 = lst.copy()

# 所有方式都是浅拷贝
print(c1 is not lst)  # True — 新对象
print(c1 == lst)       # True — 值相等
```

## 25.2 原理与机制

### 25.2.1 `==` 的 C 层面实现

`==` 运算符在 CPython 中的完整调用链：

```c
// CPython 源码: Python/ceval.c
TARGET(IS_OP) {
    PyObject *right = POP();
    PyObject *left = TOP();
    int res = (left == right);  // is 运算：直接比较指针
    // ...
}

TARGET(COMPARE_OP) {
    PyObject *right = POP();
    PyObject *left = TOP();
    PyObject *res;
    // == 运算调用 PyObject_RichCompare
    res = PyObject_RichCompare(left, right, Py_EQ);
    // ...
}
```

```c
// CPython 源码: Objects/object.c
PyObject *
PyObject_RichCompare(PyObject *v, PyObject *w, int op)
{
    // 1. 首先尝试 v 的 tp_richcompare
    PyObject *res;
    res = (*v->ob_type->tp_richcompare)(v, w, op);
    if (res != Py_NotImplemented)
        return res;
    Py_DECREF(res);

    // 2. 如果 v 返回 NotImplemented，尝试 w 的反射方法
    //    对于 ==，反射方法也是 ==
    res = (*w->ob_type->tp_richcompare)(w, v, _Py_SwappedOp[op]);
    if (res != Py_NotImplemented)
        return res;

    // 3. 如果都返回 NotImplemented，退回到默认行为
    //    对于 ==：比较 id（等价于 is）
    //    对于 !=：与 == 相反
    switch (op) {
    case Py_EQ:
        return PyBool_FromLong(v == w);  // 默认等价于 is
    case Py_NE:
        return PyBool_FromLong(v != w);
    // ...
    }
}
```

**`tp_richcompare` 在各类型中的实现**：

```c
// CPython 源码: Objects/tupleobject.c
// tuple 的 == 比较
static PyObject *
tuple_richcompare(PyObject *v, PyObject *w, int op)
{
    // 1. 类型检查
    if (!PyTuple_Check(w))
        Py_RETURN_NOTIMPLEMENTED;

    // 2. 长度检查（快速路径）
    Py_ssize_t v_len = Py_SIZE(v);
    Py_ssize_t w_len = Py_SIZE(w);
    if (v_len != w_len && (op == Py_EQ || op == Py_NE)) {
        // 长度不等时，直接返回结果
        return PyBool_FromLong((v_len == w_len) ^ (op == Py_NE));
    }

    // 3. 逐元素比较
    for (Py_ssize_t i = 0; i < Py_SIZE(v) && i < Py_SIZE(w); i++) {
        int cmp = PyObject_RichCompareBool(
            PyTuple_GET_ITEM(v, i),
            PyTuple_GET_ITEM(w, i),
            Py_EQ);
        if (cmp < 0) return NULL;  // 比较出错
        if (!cmp) {
            // 发现不等的元素
            return PyBool_FromLong(op == Py_NE);
        }
    }
    // 所有元素都相等
    return PyBool_FromLong(op == Py_EQ);
}
```

**短路优化**：CPython 对某些类型有快速路径：

```c
// CPython 源码: Objects/longobject.c
// int 的 == 比较
static PyObject *
long_richcompare(PyObject *v, PyObject *w, int op)
{
    // 快速路径：同一个对象（小整数缓存）
    if (v == w) {
        // 同一对象，直接返回结果
        switch (op) {
        case Py_EQ: Py_RETURN_TRUE;
        case Py_NE: Py_RETURN_FALSE;
        // ...
        }
    }
    // 否则逐位比较数字
}
```

### 25.2.2 `is` 的 C 层面实现

`is` 运算符比 `==` 简单得多——它直接比较两个 `PyObject *` 指针的值：

```c
// CPython 源码: Python/ceval.c
TARGET(IS_OP) {
    PyObject *right = POP();
    PyObject *left = TOP();
    int res;
    if (oparg & 1) {
        // is not
        res = (left != right);
    } else {
        // is
        res = (left == right);
    }
    SET_TOP(PyBool_FromLong(res));
    Py_DECREF(left);
    Py_DECREF(right);
    DISPATCH();
}
```

**性能差异**：

```python
import timeit

# is 比较指针：O(1)
setup_is = "a = None; b = None"
time_is = timeit.timeit("a is b", setup=setup_is, number=10_000_000)

# == 比较值：可能需要遍历整个对象
setup_eq = "a = list(range(1000)); b = list(range(1000))"
time_eq = timeit.timeit("a == b", setup=setup_eq, number=10_000_000)

print(f"is:  {time_is:.4f}s")
print(f"==:  {time_eq:.4f}s")
# is 比 == 快得多，尤其是对大型对象
```

### 25.2.3 `__hash__` 的 C 层面实现

```c
// CPython 源码: Objects/object.c
Py_hash_t
PyObject_Hash(PyObject *v)
{
    PyTypeObject *tp = Py_TYPE(v);

    // 1. 检查是否有 tp_hash 槽位
    if (tp->tp_hash != NULL)
        return (*tp->tp_hash)(v);

    // 2. 如果没有 tp_hash，检查是否定义了 __eq__
    //    如果定义了 __eq__ 但没有 __hash__，则不可哈希
    if (tp->tp_richcompare != NULL) {
        // 查找 __eq__ 方法
        PyObject *eq = PyObject_GetAttr(v, __eq_str);
        if (eq != NULL) {
            // 定义了 __eq__ 但没有 __hash__
            // 不可哈希！
            PyErr_SetString(PyExc_TypeError, "unhashable type");
            Py_DECREF(eq);
            return -1;
        }
    }

    // 3. 默认哈希：基于对象 id
    return _Py_HashPointer(v);
}
```

**内置类型的哈希实现**：

```c
// CPython 源码: Objects/tupleobject.c
// tuple 的哈希：对元素哈希值进行组合
static Py_hash_t
tuplehash(PyTupleObject *v)
{
    Py_uhash_t acc = _PyHASH_XXPRIME_5;  // 初始值
    for (Py_ssize_t i = 0; i < Py_SIZE(v); i++) {
        Py_uhash_t lane = (Py_uhash_t)PyObject_Hash(GETITEM(v, i));
        // xxHash 风格的组合
        acc += lane * _PyHASH_XXPRIME_2;
        acc = _PyHASH_XXROTATE(acc);
        acc *= _PyHASH_XXPRIME_1;
    }
    return (Py_hash_t)acc;
}

// CPython 源码: Objects/unicodeobject.c
// str 的哈希：SipHash-2-4 算法
// 使用缓存优化（hash 值存储在 PyASCIIObject.hash 中）
```

### 25.2.4 浅拷贝的 C 层面实现

```c
// CPython 源码: Objects/listobject.c
// list.copy() 的实现
static PyObject *
list_copy(PyObject *self)
{
    PyListObject *old = (PyListObject *)self;
    Py_ssize_t size = Py_SIZE(old);

    // 创建新 list 对象
    PyListObject *new = (PyListObject *)PyList_New(size);
    if (new == NULL)
        return NULL;

    // 逐个拷贝元素指针（浅拷贝！）
    for (Py_ssize_t i = 0; i < size; i++) {
        PyObject *item = old->ob_item[i];
        Py_INCREF(item);          // 引用计数 +1
        new->ob_item[i] = item;   // 拷贝指针，不拷贝对象本身
    }

    return (PyObject *)new;
}
```

关键点：**浅拷贝只拷贝指针，不拷贝指针指向的对象**。这意味着新旧列表共享相同的元素对象。

```c
// CPython 源码: Objects/dictobject.c
// dict.copy() 的实现（简化）
static PyObject *
dict_copy(PyObject *self)
{
    PyObject *new = PyDict_New();
    if (new == NULL)
        return NULL;

    // 遍历原字典的所有键值对
    // 拷贝键和值的指针（浅拷贝！）
    if (PyDict_Merge(new, self, 1) < 0) {
        Py_DECREF(new);
        return NULL;
    }

    return new;
}
```

### 25.2.5 深拷贝的 C 层面实现

`copy.deepcopy` 是纯 Python 实现，但其内部机制值得深入分析：

```python
# CPython 源码: Lib/copy.py (简化)
def deepcopy(x, memo=None):
    """深拷贝实现的核心逻辑"""
    if memo is None:
        memo = {}  # 用于处理循环引用的备忘录

    # 1. 检查备忘录（处理循环引用）
    d = id(x)
    if d in memo:
        return memo[d]  # 返回已拷贝的对象

    # 2. 获取对象的拷贝方法
    cls = type(x)
    copier = _deepcopy_dispatch.get(cls)

    if copier is not None:
        # 3. 使用类型特定的拷贝器
        y = copier(x, memo)
    else:
        # 4. 尝试使用 __deepcopy__ 方法
        if hasattr(x, '__deepcopy__'):
            y = x.__deepcopy__(memo)
        else:
            # 5. 使用 __reduce__ 或 __getstate__
            reductor = dispatch_table.get(cls)
            if reductor:
                rv = reductor(x)
            else:
                rv = x.__reduce_ex__(4)
            y = _reconstruct(x, memo, *rv)

    # 6. 存入备忘录
    memo[d] = y
    # 7. 保持备忘录与对象的生命周期同步
    _keep_alive(x, memo)

    return y
```

**备忘录（memo）的作用**：

```python
# 没有备忘录时，循环引用会导致无限递归
a = []
a.append(a)  # a 包含自身

# deepcopy 的处理过程：
# 1. 遇到 a，创建空列表 y = []
# 2. 将 y 存入 memo[id(a)] = y
# 3. 遍历 a 的元素，发现 a[0] 就是 a
# 4. 查找 memo[id(a)]，发现已经有 y
# 5. 将 y 引用自身，完成循环拷贝

import copy
b = copy.deepcopy(a)
print(b is b[0])  # True — 循环引用被正确保持
print(b is a)      # False — 新对象，不是原对象
```

## 25.3 使用场景

### 25.3.1 正确实现值对象

```python
"""值对象（Value Object）的正确实现"""
from dataclasses import dataclass
from typing import Any

@dataclass(frozen=True, eq=True)
class Money:
    """不可变金额值对象"""
    amount: int  # 以最小货币单位存储（分）
    currency: str

    def __post_init__(self):
        if self.amount < 0:
            raise ValueError("Amount cannot be negative")
        if len(self.currency) != 3:
            raise ValueError("Currency must be 3-letter ISO code")

    def add(self, other: 'Money') -> 'Money':
        if self.currency != other.currency:
            raise ValueError(f"Cannot add {other.currency} to {self.currency}")
        return Money(self.amount + other.amount, self.currency)

    def multiply(self, factor: int) -> 'Money':
        return Money(self.amount * factor, self.currency)

    def __str__(self) -> str:
        return f"{self.amount / 100:.2f} {self.currency}"

# Money 对象可以作为 dict key
price1 = Money(1000, "USD")  # $10.00
price2 = Money(1000, "USD")  # 另一个 $10.00 对象

print(price1 == price2)  # True — 值相等
print(hash(price1) == hash(price2))  # True — 哈希一致

pricing = {price1: "discounted"}
print(pricing[price2])  # "discounted" — 可以通过另一个对象找到值
```

### 25.3.2 配置对象的深拷贝

```python
"""深拷贝用于配置对象的版本管理"""
import copy

class Config:
    """可变配置对象，支持快照和回滚"""
    def __init__(self, **kwargs):
        self._data = dict(kwargs)

    def snapshot(self) -> 'Config':
        """创建深拷贝快照"""
        return copy.deepcopy(self)

    def rollback(self, snapshot: 'Config') -> None:
        """回滚到快照"""
        self._data = copy.deepcopy(snapshot._data)

    def __getattr__(self, name):
        if name.startswith('_'):
            raise AttributeError(name)
        return self._data.get(name)

    def __setattr__(self, name, value):
        if name.startswith('_'):
            super().__setattr__(name, value)
        else:
            self._data[name] = value

    def __repr__(self):
        return f"Config({self._data})"

# 使用
config = Config(host="localhost", port=8080, debug=False)
print(f"初始配置: {config}")

# 创建快照
snapshot = config.snapshot()

# 修改配置
config.host = "example.com"
config.port = 443
config.debug = True
print(f"修改后配置: {config}")

# 回滚到快照
config.rollback(snapshot)
print(f"回滚后配置: {config}")
```

### 25.3.3 实现自定义拷贝行为

```python
"""自定义 __copy__ 和 __deepcopy__"""
import copy

class Graph:
    """有向图——自定义拷贝行为"""

    def __init__(self):
        self._nodes: dict[str, set[str]] = {}

    def add_node(self, name: str) -> None:
        if name not in self._nodes:
            self._nodes[name] = set()

    def add_edge(self, from_node: str, to_node: str) -> None:
        self.add_node(from_node)
        self.add_node(to_node)
        self._nodes[from_node].add(to_node)

    def __copy__(self):
        """浅拷贝：共享节点集合"""
        new_graph = Graph()
        new_graph._nodes = self._nodes.copy()  # 浅拷贝 dict
        # 注意：集合仍然是共享的！
        return new_graph

    def __deepcopy__(self, memo: dict):
        """深拷贝：完全独立的图"""
        new_graph = Graph()
        memo[id(self)] = new_graph  # 先注册，防止循环引用
        for node, edges in self._nodes.items():
            new_graph._nodes[node] = copy.deepcopy(edges, memo)
        return new_graph

    def __repr__(self):
        lines = []
        for node, edges in sorted(self._nodes.items()):
            lines.append(f"  {node} -> {sorted(edges)}")
        return "Graph(\n" + "\n".join(lines) + "\n)"

# 创建图
g = Graph()
g.add_edge("A", "B")
g.add_edge("A", "C")
g.add_edge("B", "C")

# 浅拷贝——修改共享的集合
g_shallow = copy.copy(g)
g_shallow.add_edge("A", "D")  # 这会修改原始图的集合！
print(f"原始图（浅拷贝后）: A -> {sorted(g._nodes['A'])}")  # ['B', 'C', 'D']

# 重新创建
g = Graph()
g.add_edge("A", "B")
g.add_edge("A", "C")
g.add_edge("B", "C")

# 深拷贝——完全独立
g_deep = copy.deepcopy(g)
g_deep.add_edge("A", "D")  # 不影响原始图
print(f"原始图（深拷贝后）: A -> {sorted(g._nodes['A'])}")  # ['B', 'C']
print(f"深拷贝图: A -> {sorted(g_deep._nodes['A'])}")        # ['B', 'C', 'D']
```

```python
"""
全面分析 Python 中的比较行为
"""
import sys

def analyze_comparison():
    """分析 == 和 is 的各种行为"""

    # ---- 1. 小整数缓存 ----
    print("=== 小整数缓存与比较 ===")
    a = 256
    b = 256
    print(f"256 is 256: {a is b}")  # True（小整数缓存）

    a = 257
    b = 257
    # 注意：同一代码块中的字面量可能被编译器合并
    # 所以需要用不同方式创建
    a = int("257")
    b = int("257")
    print(f"257 is 257 (不同对象): {a is b}")  # False

    # ---- 2. 字符串驻留 ----
    print("\n=== 字符串驻留与比较 ===")
    s1 = "hello"
    s2 = "hello"
    print(f"'hello' is 'hello': {s1 is s2}")  # True（驻留）

    s3 = "hello world!"
    s4 = "hello world!"
    print(f"'hello world!' is 'hello world!': {s3 is s4}")  # 可能 True（编译器优化）

    # ---- 3. 容器类型的 == 行为 ----
    print("\n=== 容器类型的 == 行为 ===")

    # list：逐元素比较
    print(f"[1,2,3] == [1,2,3]: {[1,2,3] == [1,2,3]}")  # True

    # dict：逐键值对比较（顺序无关）
    print(f"{{'a':1,'b':2}} == {{'b':2,'a':1}}: "
          f"{{'a':1,'b':2}} == {{'b':2,'a':1}}")

    # set：元素比较
    print(f"{{1,2,3}} == {{3,2,1}}: {{1,2,3}} == {{3,2,1}}")

    # tuple：逐元素比较
    print(f"(1,2,3) == (1,2,3): {(1,2,3) == (1,2,3)}")

    # ---- 4. 不同类型的比较 ----
    print("\n=== 不同类型的比较 ===")
    print(f"1 == 1.0: {1 == 1.0}")        # True — 数值比较
    print(f"1 == True: {1 == True}")       # True — bool 是 int 的子类
    print(f"1 == '1': {1 == '1'}")          # False — 不同类型
    print(f"False == 0: {False == 0}")      # True — bool 继承自 int
    print(f"True == 1: {True == 1}")        # True

    # ---- 5. NaN 的特殊行为 ----
    print("\n=== NaN 的特殊行为 ===")
    nan = float('nan')
    print(f"nan == nan: {nan == nan}")     # False！NaN 不等于任何值
    print(f"nan != nan: {nan != nan}")      # True
    print(f"nan is nan: {nan is nan}")      # True — 仍是同一对象

    # 检查 NaN 的正确方式
    import math
    print(f"math.isnan(nan): {math.isnan(nan)}")  # True

    # ---- 6. 比较运算的链式调用 ----
    print("\n=== 链式比较 ===")
    x = 5
    # 1 < x < 10 等价于 (1 < x) and (x < 10)
    # 但 x 只计算一次
    print(f"1 < 5 < 10: {1 < x < 10}")  # True
    print(f"1 < 5 > 3: {1 < x > 3}")     # True

    # ---- 7. 比较的性能 ----
    print("\n=== 比较性能 ===")

    # is 比较：O(1)，只比较指针
    a = object()
    b = a
    t1 = timeit_compare("a is b", 10_000_000)
    print(f"is 比较 (10M次): {t1:.4f}s")

    # == 比较自定义类：调用 __eq__
    class WithEq:
        def __init__(self, x):
            self.x = x
        def __eq__(self, other):
            return isinstance(other, WithEq) and self.x == other.x

    a2 = WithEq(1)
    b2 = WithEq(1)
    t2 = timeit_compare("a2 == b2", 1_000_000)
    print(f"== 比较 (1M次): {t2:.4f}s")
```

```python
"""
深拷贝与循环引用的完整分析
"""
import copy
import weakref

# ---- 1. 简单的循环引用 ----
print("=== 简单循环引用 ===")
a = []
a.append(a)  # a 引用自身

b = copy.deepcopy(a)
print(f"b is b[0]: {b is b[0]}")  # True — 循环引用被保持
print(f"b is a: {b is a}")           # False — 新对象

# ---- 2. 双向循环引用 ----
print("\n=== 双向循环引用 ===")
class Node:
    def __init__(self, name):
        self.name = name
        self.parent = None
        self.children = []

    def add_child(self, child):
        self.children.append(child)
        child.parent = self

    def __repr__(self):
        return f"Node({self.name})"

root = Node("root")
child1 = Node("child1")
child2 = Node("child2")
root.add_child(child1)
root.add_child(child2)

# 深拷贝
root_copy = copy.deepcopy(root)
print(f"原: root.children = {root.children}")
print(f"拷: root_copy.children = {root_copy.children}")
print(f"原: child1.parent is root = {child1.parent is root}")
print(f"拷: root_copy.children[0].parent is root_copy = "
      f"{root_copy.children[0].parent is root_copy}")  # True — 循环引用正确保持

# ---- 3. 自定义 __deepcopy__ ----
print("\n=== 自定义 __deepcopy__ ===")

class DatabaseConnection:
    """模拟数据库连接——不需要深拷贝的资源"""

    _instance_count = 0

    def __init__(self, host: str, port: int):
        self.host = host
        self.port = port
        self._connected = False
        DatabaseConnection._instance_count += 1
        self._id = DatabaseConnection._instance_count

    def connect(self):
        self._connected = True

    def __deepcopy__(self, memo):
        # 数据库连接不应该被深拷贝
        # 返回同一个对象（共享连接）
        return self

    def __copy__(self):
        return self

    def __repr__(self):
        return f"DB({self.host}:{self.port}, id={self._id})"

class User:
    def __init__(self, name: str, db: DatabaseConnection):
        self.name = name
        self.db = db

    def __repr__(self):
        return f"User({self.name}, db={self.db})"

db = DatabaseConnection("localhost", 5432)
db.connect()

user1 = User("Alice", db)
user2 = copy.deepcopy(user1)

print(f"user1.db is user2.db: {user1.db is user2.db}")  # True — 共享连接
print(f"user1: {user1}")
print(f"user2: {user2}")

# ---- 4. 弱引用与拷贝 ----
print("\n=== 弱引用与拷贝 ===")

class Parent:
    def __init__(self, name):
        self.name = name
        self._children: list[weakref.ref] = []

    def add_child(self, child):
        self._children.append(weakref.ref(child))
        child.parent = self  # 注意：强引用，会导致循环引用

class Child:
    def __init__(self, name):
        self.name = name
        self.parent: Parent | None = None

parent = Parent("parent")
child = Child("child")
parent.add_child(child)

# 弱引用不会被深拷贝保持（它们指向原始对象）
parent_copy = copy.deepcopy(parent)
# 弱引用指向的对象已经不在 memo 中（除非显式处理）

# ---- 5. memo 字典的内部机制 ----
print("\n=== memo 字典分析 ===")

class Inspector:
    """检查 deepcopy 的 memo 字典"""
    def __init__(self, value):
        self.value = value

    def __deepcopy__(self, memo):
        print(f"  __deepcopy__ called, memo size: {len(memo)}")
        print(f"  memo keys (ids): {list(memo.keys())[:5]}...")
        new = Inspector(copy.deepcopy(self.value, memo))
        memo[id(self)] = new
        return new

obj = Inspector([1, Inspector([2, 3]), 4])
obj_copy = copy.deepcopy(obj)
```

### 示例3：完整的值对象体系

```python
"""
完整的值对象体系——正确实现 __eq__ 和 __hash__
"""
from dataclasses import dataclass
from typing import Any

# ============================================================
# 反面教材：不正确的 __eq__ 和 __hash__
# ============================================================

class BadPoint1:
    """错误1：定义了 __eq__ 但没有 __hash__"""
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __eq__(self, other):
        return isinstance(other, BadPoint1) and self.x == other.x and self.y == other.y

# hash(BadPoint1(1, 2)) → TypeError: unhashable type

class BadPoint2:
    """错误2：__hash__ 和 __eq__ 使用不同字段"""
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __eq__(self, other):
        return isinstance(other, BadPoint2) and self.x == other.x
        # 只比较 x，但...

    def __hash__(self):
        return hash((self.x, self.y))
        # hash 基于 x 和 y

    # 如果两个对象 x 相等但 y 不等，__eq__ 返回 True 但 __hash__ 不同
    # 这违反了 hashable 协议！

class BadPoint3:
    """错误3：__hash__ 基于可变字段"""
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __eq__(self, other):
        return isinstance(other, BadPoint3) and self.x == other.x and self.y == other.y

    def __hash__(self):
        return hash((self.x, self.y))

# 如果在 dict 中修改了 key 的字段，哈希值变化，dict 会损坏
p = BadPoint3(1, 2)
d = {p: "value"}
p.x = 99  # 修改了影响哈希的字段！
# d[p] → KeyError！因为哈希值已经变了

# ============================================================
# 正确实现：使用 frozen dataclass
# ============================================================

@dataclass(frozen=True)
class Point:
    """不可变点——正确的 __eq__ 和 __hash__"""
    x: float
    y: float

    def distance_to(self, other: 'Point') -> float:
        return ((self.x - other.x) ** 2 + (self.y - other.y) ** 2) ** 0.5

    def __neg__(self) -> 'Point':
        return Point(-self.x, -self.y)

    def __add__(self, other: 'Point') -> 'Point':
        return Point(self.x + other.x, self.y + other.y)

    def __sub__(self, other: 'Point') -> 'Point':
        return Point(self.x - other.x, self.y - other.y)

    def __mul__(self, scalar: float) -> 'Point':
        return Point(self.x * scalar, self.y * scalar)

# frozen=True 自动生成 __hash__ 和 __eq__
p1 = Point(3.0, 4.0)
p2 = Point(3.0, 4.0)
p3 = Point(1.0, 2.0)

print(f"p1 == p2: {p1 == p2}")          # True — 值相等
print(f"p1 is p2: {p1 is p2}")           # False — 不同对象
print(f"hash(p1) == hash(p2): {hash(p1) == hash(p2)}")  # True

# 可以安全地用作 dict key
point_map = {
    p1: "A点",
    p3: "B点",
}
print(point_map[p2])  # "A点" — p2 可以找到 p1 的值

# 可以安全地用作 set 元素
point_set = {p1, p2, p3}
print(f"集合大小: {len(point_set)}")  # 2 — p1 和 p2 是同一个元素

# ============================================================
# 更复杂的值对象
# ============================================================

@dataclass(frozen=True)
class Money:
    """不可变金额"""
    amount: int  # 最小货币单位
    currency: str

    def __post_init__(self):
        if self.amount < 0:
            raise ValueError("Amount cannot be negative")
        if len(self.currency) != 3:
            raise ValueError("Currency must be 3-letter code")

    def add(self, other: 'Money') -> 'Money':
        if self.currency != other.currency:
            raise ValueError(f"Currency mismatch: {self.currency} vs {other.currency}")
        return Money(self.amount + other.amount, self.currency)

    def __str__(self):
        major = self.amount // 100
        minor = self.amount % 100
        return f"{major}.{minor:02d} {self.currency}"

# Money 对象可以作为 dict key
m1 = Money(1000, "USD")
m2 = Money(1000, "USD")
m3 = Money(2000, "USD")

prices = {m1: "ten dollars", m3: "twenty dollars"}
print(prices[m2])  # "ten dollars"

# ============================================================

@dataclass(frozen=True)
class Rectangle:
    """不可变矩形"""
    width: float
    height: float

    @property
    def area(self) -> float:
        return self.width * self.height

    @property
    def perimeter(self) -> float:
        return 2 * (self.width + self.height)

    def contains(self, point: Point) -> bool:
        return 0 <= point.x <= self.width and 0 <= point.y <= self.height

@dataclass(frozen=True)
class Circle:

    center: Point
    radius: float

    @property
    def area(self) -> float:
        import math
        return math.pi * self.radius ** 2

    def contains(self, point: Point) -> bool:
        return self.center.distance_to(point) <= self.radius

# 组合值对象也是可哈希的
```

## 25.5 常见陷阱与最佳实践

### 陷阱1：`==` 与 `is` 的混淆

```python
# 错误：用 == 比较单例
x = None
if x == None:    # 可以工作但不推荐
    pass

# 正确：用 is 比较单例
if x is None:    # PEP 8 推荐
    pass

# 错误：用 is 比较值
a = 257
b = 257
if a is b:       # 可能 False！超出小整数缓存范围
    pass

# 正确：用 == 比较值
if a == b:       # 始终正确
    pass

# 特殊情况：NaN
nan = float('nan')
nan == nan       # False — NaN 不等于任何值
nan is nan       # True — 是同一个对象

# 检查 NaN 的正确方式
import math
math.isnan(nan)  # True
```

### 陷阱2：修改作为 dict key 的对象

```python
# 错误：使用可变对象作为 dict key
class BadKey:
    def __init__(self, x):
        self.x = x

    def __eq__(self, other):
        return isinstance(other, BadKey) and self.x == other.x

    def __hash__(self):
        return hash(self.x)  # 哈希值依赖于可变字段！

k = BadKey(1)
d = {k: "value"}
k.x = 2  # 修改了 key！
# d[k] → KeyError！哈希值已变

# 正确：使用不可变对象作为 key
@dataclass(frozen=True)
class GoodKey:
    x: int  # frozen=True 保证不可变

k = GoodKey(1)
d = {k: "value"}
# k.x = 2  → FrozenInstanceError！阻止修改

# 替代方案：使用 tuple 或 frozenset 作为 key
key = (1, 2, 3)
d = {key: "value"}
```

### 陷阱3：浅拷贝导致意外共享

```python
# 陷阱：浅拷贝只拷贝一层
original = {"names": ["Alice", "Bob"], "config": {"debug": True}}
shallow = original.copy()

shallow["names"].append("Charlie")
print(original["names"])  # ['Alice', 'Bob', 'Charlie'] — 被修改了！

deep["names"].append("David")

original = {"names": ("Alice", "Bob"), "config": MappingProxyType({"debug": True})}
# names 是 tuple，不可变
# config 是 MappingProxyType，只读

# 解决方案3：使用 frozen dataclass
@dataclass(frozen=True)
class Config:
    debug: bool
    names: tuple  # 使用 tuple 代替 list
```

### 陷阱4：`__eq__` 返回 `NotImplemented` vs `NotImplementedType`

```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __eq__(self, other):
        # 错误：返回 False 而非 NotImplemented
        if not isinstance(other, Vector):
            return False  # 这会阻止反向比较！

    # 正确：返回 NotImplemented
    def __eq__(self, other):
        if not isinstance(other, Vector):
            return NotImplemented  # 让 Python 尝试 other.__eq__(self)
        return self.x == other.x and self.y == other.y

# 返回 NotImplemented 时的行为：
# Vector(1, 2) == (1, 2)
# → Vector.__eq__((1,2)) → NotImplemented
# → tuple.__eq__((1,2), Vector(1,2)) → False（tuple 不认识 Vector）
# → 最终结果：False

# 返回 False 时的行为：
# Vector(1, 2) == (1, 2) → False（直接返回，不尝试反向）
# 但这可能在继承场景中导致问题
```

### 陷阱5：深拷贝的性能陷阱

```python
import copy
import time

# 深拷贝大型对象可能很慢
class BigData:
    def __init__(self, size):
        self.data = list(range(size))

    def __deepcopy__(self, memo):
        # 自定义深拷贝可以优化性能
        start = time.time()
        new = BigData(0)
        new.data = self.data.copy()  # 只需要浅拷贝（元素都是 int）
        return new

big = BigData(1_000_000)

# 标准 deepcopy
start = time.time()
_ = copy.deepcopy(big)
print(f"标准 deepcopy: {time.time() - start:.4f}s")

# 自定义 deepcopy
start = time.time()
_ = big.__deepcopy__({})
print(f"自定义 deepcopy: {time.time() - start:.4f}s")
# 自定义版本可能快得多（如果内部元素不需要深拷贝）
```

### 最佳实践总结

| 场景 | 推荐做法 | 原因 |
|------|----------|------|
| 与 `None` 比较 | 用 `is` | PEP 8 推荐，语义清晰 |
| 与 `True`/`False` 比较 | 用 `is` | 单例对象 |
| 值比较 | 用 `==` | 比较内容而非身份 |
| 定义值类型 | 用 `@dataclass(frozen=True)` | 自动生成正确的 `__eq__` 和 `__hash__` |
| `__eq__` 不支持类型 | 返回 `NotImplemented` | 让 Python 尝试反向比较 |
| `__hash__` 定义 | 基于 `__eq__` 使用的相同字段 | 保证 `a == b → hash(a) == hash(b)` |
| dict key | 使用不可变类型 | 避免哈希值变化导致数据丢失 |
| 浅拷贝 | 用 `.copy()`、`[:]`、`dict()` | 只需拷贝一层时 |
| 深拷贝 | 用 `copy.deepcopy()` | 需要完全独立副本时 |
| 大型对象拷贝 | 自定义 `__deepcopy__` | 避免不必要的递归拷贝 |
| 循环引用 | `deepcopy` 自动处理 | memo 字典防止无限递归 |
| 资源对象 | `__deepcopy__` 返回自身 | 连接、文件句柄等不应拷贝 |
| 检查 NaN | 用 `math.isnan()` | NaN 不等于自身 |
